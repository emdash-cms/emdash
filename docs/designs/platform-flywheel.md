## Vision

### 10x Check
EmDash as the AI-native publishing platform. Not just AI-assisted writing, but AI that understands your entire content strategy. Natural language command palette ("Show me all draft posts from last week"), content strategy analysis, auto-generated SEO metadata, internal linking suggestions. The command palette becomes a conversational interface to the entire CMS.

### Platonic Ideal
A CMS where:
- The editor feels like writing in a beautifully designed notebook. No UI friction between thought and published page.
- AI is invisible infrastructure, not a feature. The CMS predicts what you need before you ask.
- Plugins are one-click install and curated like the App Store.
- Migrating from WordPress is a single command that preserves everything, including SEO rankings.
- Visual editing lets you click any element on your live site and edit inline.
- Live preview updates in real-time as you type.
- Your content is portable. Export everything, any format, any time.

## Revised Phase Structure (post outside voice)

### Phase 1 (Week 1-2): WordPress Migration + Ghost-Killer Editor
- WordPress migration wizard polish + SEO preservation (MOVED UP from Phase 3)

### Phase 2a (Week 3-5): Admin Renaissance Core
- Command palette (Cmd+K) with fuzzy search (NOT NL yet)

### Phase 2b (Week 6-8): AI + Preview
- Shared AI service abstraction (emdash.ai)

### Phase 2c (Week 9-11): Visual Editing + Polish
- Visual editing completion (click-to-edit on live site)

- Accessibility audit (WCAG 2.1 AA)

### Phase 3 (Week 12+): Ecosystem & Growth
- Plugin/theme starter kits
- Marketplace curation + WordPress-equivalent mapping
- NL command palette (moved from Phase 2)

- First-party SEO plugin
- Developer docs, landing page, plugin bounties

## Key Decisions from Outside Voice
1. **TipTap permanently.** No BlockNote evaluation. Commit to the existing editor stack.
2. **WordPress migration moved to Phase 1.** It's the acquisition funnel, not an afterthought.
3. **Phase 2 split into 3 sub-phases.** 10+ features in 6 weeks was overloaded.
4. **NL command palette deferred to Phase 3.** Fuzzy search first, NL later.
5. **Production live preview via on-demand SSR.** Not dev-only.
