Improves WordPress plugin imports: taxonomy terms and assignments, Yoast/Rank Math SEO titles and descriptions, ACF values with matching schema fields, and navigation menus are now imported. Fetches the full media library instead of the first 500 files, supports sites with plain permalinks via a `?rest_route=` fallback, and the import screen accepts a migration key generated by the EmDash Exporter plugin wizard.
